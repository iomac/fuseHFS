# fuseHFS
A macFUSE filesystem built on top of hfsutils/libhfs to enable working with classic 
Macintosh Hierarchial File System (HFS) HFS on modern Macs.

This project is based on the LoopbackFS macFUSE example project.

# Prerequisites
1. maxFuse 4.2.4+
2. Xcode 13+
3. macOS 12+
